So we need a way to make data arrive at the end layers to make the output more meaningful when the
weights are still random, and we need to propagate the proper gradients properly backwards into the start
before theyre turned into random noise

through them normally but also pass around them without being processed ^3UkrgyJ1

At the end of every block we can combine the input to the block with
the output of the block element-wise or concatenate them (addition is preferred to minimize the
tensor size growing very large) ^s7WGLd5Q

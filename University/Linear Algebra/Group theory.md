# Text Elements
Group ^iVeRtccp

A set g, together with an operation *:gxg -> g
is called a Group denoted by G(g,*) if all of the
following hold: ^lIUTJVxW

1. The operation * is associative
(a*b)*c = a*(b*c)

2. There exists a unique identity element in the group
x*i = i*x = x

3. There exists a unique inverse element in the group for every element in the group
a * a = a * a = i ^mQbW6tkB

-1 ^eyyRqAio

-1 ^CJbIPbgM

The group is also called commutative
if the operation * is also commutative
a*b = b*a ^bxk4bHaw

Field ^iXvjPNZn

A set f, together with operations + and x is called a field,
denoted by F = (f,+,x) if all of the following properties hold: ^cJeHFhcW

1. (f, +) Form a commutative group
2. (f-{0}, x) form a commutative group
3. these operations are distributive
a x (b+c) = (axb) + (axc) ^wWBeqJCT

a ring is a Field
that does not require its multiplication
operation to be invertible or have
a unique identity element ^2tzZE2Jx
